### Run Locally

To run **Img-Slider** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/alphaotuken/Img-Slider.git
```

Windows:

```bash
git clone https://github.com/alphaotuken/Img-Slider.git
```
